KEY LEARNINGS & SOLUTIONS
================================================================================

1. MySQL DECIMAL fields return as strings in JSON
   → Solved with parseFloat() conversion in backend API responses

2. Network binding requires 0.0.0.0, not localhost
   → Required for multi-device access across network

3. Vite requires explicit host configuration
   → Added server: { host: '0.0.0.0', port: 5173 } to vite.config.js

4. Chart data type matters for Recharts
   → Recharts needs numbers, not strings for proper rendering

5. Consistent formatting improves UX
   → Implemented formatAmount, formatAmountShort, formatNumber throughout
